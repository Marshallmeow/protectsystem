Получение сведений о системе

Цель работы

Получить сведения об используемой системе

Ход работы

  1. Получение сведений о версии ядра:

    uname -r
5.10.0-9-amd64

  1. Получение всех сведений о ядре:

    uname -a
    Linux debian11 5.10.0-21-amd64 #1 SMP Debian 5.10.162-1 (2023-01-21) x86_64 GNU/Linux

  2. Получение сведений об используемом дистрибутиве:

    sudo lsb_release -a
    Distributor ID: Debian
Description:    Debian GNU/Linux 11 (bullseye)
Release:    11
Codename:   bullseye

  3. Получение данных о модели процессора:

    cat /proc/cpuinfo | grep "model name"
    model name  : Intel(R) Core(TM) i7-10750H CPU @ 2.60GHz
model name  : Intel(R) Core(TM) i7-10750H CPU @ 2.60GHz
